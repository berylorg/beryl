# Wishlist

- The graph overlay should eventually support full keyboard traversal across explorer columns, including moving between sibling rows and stepping left or right through the active column history without leaving the keyboard.
- The workspace picker popup should eventually support keyboard-first switching, creation, and deletion flows across existing workspaces without requiring pointer use.
- The transcript should eventually expose keyboard navigation between major backend item blocks inside a long thread, beyond the conversation composer's turn-boundary jumps.
- The reusable code widget should eventually provide language-labeled, parser-backed syntax highlighting through one shared app-wide highlighter while continuing to accept plain-text source as input.
